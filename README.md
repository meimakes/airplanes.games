# ✈️ airplanes.games

Playful learning for future pilots. 5 educational games designed for ages 3–5 that build pattern recognition, logic, spatial reasoning, counting, and problem-solving skills — all through airplanes.

## Games

- **Landing Path** — Trace the glide path to guide your plane safely to the runway (Spatial Reasoning)
- **Pattern Lab** — Spot the repeating sequence and drag the right plane to complete it (Pattern Recognition)
- **Logic Hangar** — Match each airplane to its hangar by looking carefully at the signs (Matching & Sorting)
- **Flight Path** — Navigate a maze while dodging storm clouds to reach the runway (Problem Solving)
- **Cargo Count** — Count the cargo boxes as they load up, then tap the right number (Counting)

## Tech

Zero dependencies. Pure HTML/CSS/JS. Touch-friendly. Mobile-first. Works in any browser.

Each game uses real airplane images, Web Audio API sound effects, and confetti celebrations.
